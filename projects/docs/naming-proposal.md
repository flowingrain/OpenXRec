# 智弈全域风险态势感知平台 - 命名方案

## 最终命名

| 项目 | 内容 |
|------|------|
| **软件全称** | 智弈全域风险态势感知平台软件 |
| **软件简称** | 智弈态势分析平台 |
| **英文名称** | Zhiyi Pan-Domain Risk Situation Awareness Platform |

---

## 命名解析

### "智弈"的含义

- **智**：智慧、智能，体现人工智能技术驱动
- **弈**：博弈、决策，体现风险态势的推演和决策能力
- **智弈**：智能博弈，寓意平台以AI驱动复杂的态势分析与决策

### "全域"的含义

覆盖宏观、中观、微观三个层级的完整风险态势感知：

```
全域 = 宏观层（政策/行业/区域）
     + 中观层（产业链/供应链）
     + 微观层（企业/项目/资产）
```

### "风险态势感知"的含义

- **风险**：聚焦风险识别、评估、预警
- **态势**：动态的、演化的状态和趋势
- **感知**：感知、认知、理解的能力

---

## 核心价值定位

### 双视角联动能力

| 视角 | 分析内容 | 典型场景 |
|------|---------|---------|
| **宏观视角** | 政策趋势、行业周期、区域经济、地缘政治 | 油价分析、货币政策影响、地缘风险评估 |
| **微观视角** | 企业经营、财务健康、信用风险、项目评估 | 上市公司风险分析、企业信用评估 |
| **联动分析** | 宏观环境对微观主体的影响传导 | 行业政策变化对企业的影响预测 |

### 典型应用场景

1. **分析某上市公司风险** → 自动关联行业趋势 + 政策环境
2. **预测行业变化** → 自动提示受影响的重点企业
3. **油价分析** → 从地缘政治到产业链再到企业影响
4. **政策影响评估** → 从宏观政策到微观主体的传导路径
