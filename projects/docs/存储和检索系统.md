## 三、各组件职责定义

### 1. 数据库 - 元数据与事务层

| 职责 | 说明 | 示例 |
|------|------|------|
| **用户数据** | 用户画像、偏好设置 | `user_profiles` |
| **关系数据** | 实体关系、社交图谱 | `kg_relations` |
| **事务数据** | 反馈、评分、交互记录 | `feedbacks`, `interactions` |
| **配置数据** | 系统配置、策略配置 | `ppo_hyperparam_versions` |
| **日志审计** | 操作日志、变更历史 | `audit_logs` |

**特点**：强一致性、ACID事务、复杂JOIN查询

### 2. 知识库 - 事实知识层

| 职责 | 说明 | 示例 |
|------|------|------|
| **结构化知识** | 实体定义、属性、分类 | "腾讯是一家中国科技公司" |
| **关系知识** | 实体间的关系 | "腾讯投资了京东" |
| **规则知识** | 业务规则、逻辑规则 | "股价下跌超过10%触发预警" |
| **概念知识** | 领域概念、术语解释 | "什么是RAG" |

**特点**：
- 以**三元组**（实体-关系-实体）为核心
- 支持**图查询**（路径、邻居、子图）
- 置信度标记、来源追溯
- 版本管理、冲突检测

### 3. 案例库 - 经验知识层

| 职责 | 说明 | 示例 |
|------|------|------|
| **完整案例** | 历史推荐案例 | "某用户询问AI趋势，系统推荐了..." |
| **分析模板** | 可复用的分析框架 | "竞品对比分析模板" |
| **用户案例** | 用户生成的案例 | 用户保存的分析报告 |
| **失败案例** | 错误示范、边界情况 | "低质量推荐案例" |

**特点**：
- 以**完整上下文**为单位
- 包含查询、推理过程、结果、反馈
- 支持**相似案例检索**
- 质量评分、标签分类

### 4. 向量库 - 语义索引层

| 职责 | 说明 | 示例 |
|------|------|------|
| **语义检索** | 相似内容检索 | 查找与查询语义相似的知识/案例 |
| **聚类分析** | 内容自动分类 | 发现知识的主题聚类 |
| **去重检测** | 相似内容检测 | 判断新知识是否已存在 |
| **推荐召回** | 候选集召回 | 第一阶段的大规模召回 |

**特点**：
- **不存储原始内容**，仅存储向量+ID引用
- 高性能**近似最近邻(ANN)**检索
- 支持**多模态**（文本、图像、视频）

### 5. 在线检索 - 实时数据层

| 职责 | 说明 | 示例 |
|------|------|------|
| **实时新闻** | 最新发生的事件 | "今天腾讯发布了..." |
| **动态数据** | 实时变化的数据 | 股价、汇率、天气 |
| **时效性查询** | 需要最新信息的查询 | "2024年最新的AI发展" |
| **未知领域** | 知识库中不存在的领域 | 新出现的产品/公司 |

**特点**：
- **每次实时查询**，不持久化
- 依赖外部搜索引擎
- 返回结果需要**可信度评估**
- 可选择性**固化**到知识库

## 四、在线检索 vs RAG 对比

| 维度 | 在线检索 | RAG (知识库+向量库) |
|------|-------------------|---------------------|
| **数据来源** | 互联网搜索引擎 | 内部知识库、案例库 |
| **时效性** | 实时（秒级） | 延迟（天/周级） |
| **可控性** | 低（依赖外部） | 高（自主管理） |
| **准确性** | 参差不齐 | 经过验证固化 |
| **成本** | 每次查询付费 | 一次性存储+检索成本 |
| **隐私性** | 数据外泄风险 | 数据内网存储 |
| **适用场景** | 实时信息、未知领域 | 领域知识、历史案例 |
| **可解释性** | 来源URL | 知识来源+置信度 |

### 协作流程

```
用户查询
    ↓
┌───────────────────────────────────────────────────────────┐
│                    意图分析                                │
│  判断查询类型：实时/历史/混合                               │
└───────────────────────────────────────────────────────────┘
    ↓
┌───────────────────────────────────────────────────────────┐
│                    检索策略选择                            │
│                                                           │
│  实时类查询 ──────────→ 在线检索 → 返回结果                │
│  （"今天新闻"）                ↓                          │
│                          高价值内容 → 固化到知识库          │
│                                                           │
│  历史类查询 ──────────→ RAG检索 → 返回结果                 │
│  （"什么是RAG"）               ↓                          │
│                          无相关知识 → 在线检索补充          │
│                                                           │
│  混合类查询 ──────────→ RAG + 在线检索 → 融合结果          │
│  （"最新AI发展趋势分析"）                                  │
│                                                           │
└───────────────────────────────────────────────────────────┘
```
